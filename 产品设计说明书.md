Xmind —— 离线可用功能需求说明书（产品设计说明）

1. 产品背景

作为一款思维导图工具，Xmind 必须支持在无网络环境下稳定运行。用户在地铁、飞机、会议室或弱网络区域依然能够创建、编辑与保存脑图内容。因此，需要确认哪些核心功能在纯离线环境中保持可用。

2. 离线可用功能范围
   2.1 文件创建与管理

新建导图文件

支持离线创建空白导图。

支持从本地模板创建导图。

文件打开与保存

打开本地 .xmind 文件。

自动保存（离线状态不影响）。

支持“另存为”到本地。

本地版本管理

自动保留历史版本（本地数据库或临时文件）。

崩溃恢复文件在本地生成和读取。

导入文件

导入本地 Xmind、MindManager、Markdown、Txt 等文件。

导出文件

导出为 PNG、PDF、Markdown、Txt（无需网络）。

2.2 思维导图核心编辑功能

节点操作

新增主题、子主题、同级主题。

删除、复制、粘贴、剪切主题。

拖动节点调整结构。

结构类型

逻辑结构树（中心向外）

鱼骨图

组织结构图

思维结构图

时间线、矩阵（如果产品本身支持）

样式编辑

修改节点颜色、字体、背景、形状。

批量样式复制（格式刷）。

主题备注

文本备注、超链接（允许但离线不可跳转外链）。

图标与贴纸

离线本地图标库可直接使用。

图片

插入本地图片。

基本图片缩放与布局。

2.3 离线资源库支持

本地模板库

所有交付安装包自带的模板可离线使用。

本地图标库 / 贴纸库

用户可在无网络环境中浏览与插入。

本地主题样式

保存自定义样式到本地。

字体离线可用

本地系统字体 / 软件自带字体正常渲染。

2.4 交互与体验

撤销/重做（Undo/Redo）

本地状态栈，无网络依赖。

键盘快捷键

主题操作、样式调整、文件功能快捷键均离线可用。

搜索功能（本地）

对当前导图内容搜索关键词。

2.5 离线导图表现功能

大纲视图（Outline）

支持导图与大纲双向编辑。

多页导图（Sheet）切换

多视图切换

思维导图视图

大纲视图

展开/收起节点

2.6 多媒体能力（离线限制说明）

插入附件（本地文件）

支持插入 PDF、Doc、图片等作为附件链接。

插入音频（本地文件）

可播放本地存储的音频记录。

录音功能（如果产品支持）

允许在无网环境下录制语音并保存到本地。

3. 离线不可用或受限功能说明
   3.1 云相关功能

云端文档同步（Xmind Cloud）

在线协作、多人编辑

跨设备同步

在线模板库（网络模板无法加载）

在线图标库（需网络下载的图标不可使用）

3.2 AI 功能

AI 拓展主题

AI 总结

AI 自动排版优化
→ 所有基于网络的 AI 功能不可用。

3.3 在线导出 / 在线服务依赖

在线 PDF 转换（如某些老版本使用云服务）

分享链接生成

4. 离线状态下的错误提示设计

弱网或无网络检测

本地保存时不提示。

云同步时显示“当前处于离线状态，已自动切换为本地模式。”

点击在线功能时

以非阻塞方式展示提示：
“该功能需要网络，当前离线无法使用。”

资源加载失败

模板、贴纸加载失败提示应给用户备选方案（如本地版本）。

5. 离线模式下的数据安全要求

本地文件加密（可选）

提供密码保护功能。

本地缓存位置

自动保存与历史版本写入本地缓存目录。

断网时编辑不丢失

自动保存间隔与在线模式一致。

离线恢复机制

崩溃重启后提示恢复上次未保存内容。

6. 性能与兼容性要求

离线环境不影响启动速度

软件应在无网环境下秒级启动。

离线编辑大文件稳定性

大型导图（>2000 节点）仍能正常编辑。

跨平台一致性

Windows / Mac / Linux 离线功能一致。

7. 可扩展离线功能（建议项）

离线全文检索增强版本

对多个本地导图进行搜索。

本地插件体系（类似 VSCode）

插件完全本地运行，不依赖网络。

离线自动排版算法（无需 AI）

提供本地计算的智能布局。
